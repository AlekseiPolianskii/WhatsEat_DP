1. Требуется скачать интерпретатор Python (https://www.python.org/downloads/release/python-397/)
или IDE (PyCharm)
2. загрузить все необходимые пакеты из файла requirements.txt (команда pip install -r requirements.txt)
3. обновить структуру БД согласно файлу my_db.sql
4. запустить файл parserVER_02.py

Должна наполниться картинками папка img ('../src/main/resources/static/sample/img/'), таблицы нашей БД whatseat.
Файл V3_INSERT_VALUES.sql в папке migration ('../src/main/resources/db/migration/V3_INSERT_VALUES.sql') должен дублировать все SQL-запросы на заполнение.
